# fastprint-test
## Installation

Start with cloning this repo on your local machine:

```sh
$ git clone https://github.com/adehusnim37/fastprint-test.git
```

To install and set up the library, run:

```sh
$ cd backend
$ npm install
```

```sh
$ cd frontend
$ npm install
```

## Usage

### Serving the app

```sh
$ cd backend
$ npm start
```
